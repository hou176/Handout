Git单人本地仓库操作--安装配置

  • 提示：本地仓库是个.git隐藏文件

以下为演示Git单人本地仓库操作

  • 1.安装git

      sudo apt-get install git
  密码：chuanzhi

  • 2.查看git安装结果

      git

  • 3.创建项目

    • 在桌面创建test文件夹，表示是工作项目

        Desktop/test/

  • 4.创建本地仓库

    • 进入到test，并创建本地仓库.git

    • 新创建的本地仓库.git是个空仓库

        cd Desktop/test/
  git init

    • 创建本地仓库.git后

  • 5.配置个人信息

      git config user.name '张三'
  git config user.email 'zhangsan@163.com'

    • 配置个人信息后

默认不配置的话，会使用全局配置里面的用户名和邮箱 全局git配置文件路径：~/.gitconfig
