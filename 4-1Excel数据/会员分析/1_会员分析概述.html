一. Excel的基本使用

会员分析概述

指导理论: AIPL

  • 认知(Aware) -> 兴趣(Interest) -> 购买(Purchase) -> 忠诚(Loyalty)
  • 会员运营的四个阶段:
    • 拉新: 获取新用户
    • 促活: 兴趣(Interest) -> 注册会员
    • 转换: 购买(Purchase)
    • 留存: 忠诚(Loyalty) -> 复购

目标

通过会员分析, 为会员运营提供支持, 从而提高销量和利润

  • 会员等级说明
    • 白银: 注册(0)
    • 黄金: 下单(1~3888)
    • 铂金: 3888~6888
    • 钻石: 6888以上

会员存量与增量分析

通过会员存量与增量分析, 来观察会员的运营情况, 为会员运营提供决策的数据依据.

  • 源数据

    • 会员信息查询.xlsx, 从中台系统导出, 需额外添加年月字段，方便统计
    • 门店信息表.XLSX, 主要用到地区编码这个字段

  • 整体概览

    • 存量增量分析

      • 目的: 查看会员拉新情况

    • 增量等级分布

      • 目的: 查看会员拉新与转换情况

    • 增量等级占比分析

      • 目的: 查看会员转换情况, 是否有发生购买行为

    • 整体等级分布

      • 目的: 查看整体会员的分布情况
      • 电商与线下存量增量分析
        • 目的: 查看电商与线下会员增量情况

    • 地区店均会员分析

      • 由于每个地区的店铺数量不一样，看会员总量难以对比各个地区的会员存量，所以使用店均会员数量进行对比
      • 目的: 查看每个地区会员存量情况

    • 会员明细表

      • 门店渠道会员
        • 作用：为门店做会员运营, 提供数据支撑. 如会员生日、门店上新和促销等

会员消费分析

  • 源数据

    • 会员消费报表.xlsx, 会员消费记录
    • 门店信息表.XLSX, 主要用到地区编码这个字段
    • 全国销售订单数量表.xlsx, 用作会员的占销比计算

  • 各个地区会销比

    • 会销比 = 会员消费的订单数 / 全部销售订单数
      • 原会销比 = 会员消费的金额 / 全部客户消费的金额
      • 但是由于金额无法获取，所以使用订单数取代，也有公司使用订单数计算的
    • 目的: 衡量会员消费的价值
  • 连带率
    • 连带率 = 消费数量 / 订单数量
    • 通过提高连带率, 可以提高客单价, 从而提高销售金额 和 利润率
    • 目的: 查看每一个店铺的连带率.
  • 复购率
    • 根据企业主要业务选取时间段计算，这里选取一年作为复购时间计算复购率；
    • 一个会员同一天消费多次也算一次消费，所以会员消费次数按一天一次计算
    • 复购率 = 一年内消费次数大于等于2次的人数 / 全部人数
